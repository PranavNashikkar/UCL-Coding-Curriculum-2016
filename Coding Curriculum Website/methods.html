5. Methods



Methods are behaviours of classes that are used to execute certain operations, and they have a certain syntax they have to follow to allow the compiler to interpret them.


5.1 Method Syntax


A method is defined by including the following in the same order:

  1. Access type – This determines if the method can be accessed outside the class it is in.
    • Public – This means everything can access this method in this class. Visible to everything.
    • Private – This means only methods from inside the class can access this method. Visible to only class.
  2. Return data type – Methods are allowed to return a value but this must be defined in the method declaration as the data type of the value to be returned.
    • Data type – This can be an int, String, char, float etc. However, if you choose to have a return value, the method will EXPECT a return statement alongside a variable of the data type inside it.
    • void – If your method is not required to return anything, you can use the void key word which indicated that this method will not return a value.
  3. Name – This is simple and just identifies what the name of the method will be. It must not be a Java key word.
  4. Parameters – these are values that you can pass into a method as arguments. In Java, parameters are passed by value only.
    • Passing by value – This means that after the method is run, any calculation on the parameters passed do not change the original values.

Example of methods that use syntax rules above in different ways:

public static void main(String args[]){
int a = 10;
int b = 5;
int answer=returnSum(a,b);
System.out.println("The sum is "+answer);
}

public static int returnSum(int a, int b){
int sum = a+b;
return sum;
}

Above we can see that the returnSum method returns a value of data type int, and that it takes 2 integer parameters a and b, and returns the sum of both.


Example of a method without a return value:

public static void main(String args[]){
returnSum();
}

public static int returnSum(){
int a = 10;
int b = 5;
int sum = a + b;
System.out.println("The sum is "+sum);
}

As we can see the program above achieves the same output as the first yet this one does not take parameters or return a value.
