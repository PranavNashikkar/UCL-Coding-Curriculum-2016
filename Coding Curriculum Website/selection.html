6. Selection



Selection refers to the decision making concept in programming. This means a certain condition is considered and if it happens to be true, a certain operation occurs. The opposite can also happen, so if it is false, another action occurs.


6.1 IF Statement

In Java this is best done by an IF statement.

Below is a flow chart that visually displays the concept of an IF selection statement and how it works. A boolean expression or condition is expressed and depending on the outcome, a certain action will be performed but not both.


The syntax of an if statement is quite simple, it consists of a boolean expression to be evaluated. If the statement is true, the first block of code is run, otherwise if false you can use the Java key word else to run another block of code. These blocks of code must be separated by curly braces.

Here is an example:

String output;
int testNumber=5;

if(testNumber==5){
output="The condition is true";
}else{
output="The condition is false";
}

System.out.println(output);

In the example above the if statement evaluates whether the integer variable testNumber is 5, and if so, the output will be “The condition is true”, otherwise if false, the output will be “The condition is false”.

6.2 SWITCH Statement


However, if you have many conditions that might need evaluating, multiple if statements can be tedious to write and hard to read. This is where the switch statement comes in. This allows you to evaluate multiple conditions in one block of code.

A single variable can be evaluated against many values so the argument passed into the switch statement is the variable being tested.

  • Case key words identify what happens when there is equality between the argument variable and a chosen condition. Each case block of code must include a colon after the boolean expression.

  • Break key word tells the loop/selection block of code to exit the loop/selection statement if it is reached.

  • Default key word identifies what happens when none of the boolean expressions are evaluated to be true.

Below is a flow chart that visualises how the switch statement works:

Here is an example block of code for a SWITCH statement: 

String output;
int testNumber=5;

switch(testNumber){
case 1:
	output="The number is 1";
	break;
case 2:
	output="The number is 2";
	break;
case 3:
	output="The number is 3";
	break;
default:
	output="The number is not 1,2, or 3";
	break;
}

System.out.println(output);
}
