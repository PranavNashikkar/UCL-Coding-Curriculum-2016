4. Arrays



An Array is a data structure which contains a fixed or static number of elements of identical data type.


4.1 Declaration, Instantiation, and Indexing

Declaration

String[] strArray;
int[] intArray;

Instantiation

String[] strArray=new String[10];
int[] intArray=new int[50];

int arrayLength=10;
String[] strArray=new String[arrayLength];

This is the common way of creating an array. It makes use of the Java keyword new and identifies the number of elements being allocated to this array in the square brackets. It can use a fixed value 10 for example, or another integer variable and the value stored in it.

However, Java arrays can also be created with literals too.

String[] strArray={"Hello", "World", "!"};;
int[] intArray={2,5,7,3,4,6,45};

In this example each array is defined not with the number of elements but rather the elements themselves. The String array will contain 3 elements and the integer array will contain 6 elements.

Indexing

Above is an example of printing the elements of the array by using the index reference in the square brackets.



4.2 Useful Array Object Methods


Since arrays are objects they have useful methods that allow you to do cool things with them. Here is one:

  • length() – This method allows you to calculate the length of the array, meaning the number of elements it has. The reason why this is useful is because you might have used another integer variable to determine the array’s size. So since the integer variable could change during runtime, you might need to find the length of the array. Below is an example of it in use.
int arrayLength=16;
String[] strArray=new String[arrayLength];
System.out.println("The length of this array is "+strArray.length);
