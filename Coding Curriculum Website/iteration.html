7. Iteration



In some conditions you might want a certain block of code to run many times or a fixed number of times depending on the output from your program. The best way to do this is to implement iteration.

Below is a flowchart that presents iteration visually:

 

This flowchart basically explains that while a certain condition is met, that particular block of code will continue running repeatedly until the condition is no longer met.


In Java there are three main loop constructs and they are: FOR, WHILE, and DO WHILE.


7.1 FOR Loops


A FOR loop is a great loop construct to use if you know exactly how many times you need your block of code to run.

Syntax

When you declare a FOR loop there are three parts of code that need to be addressed:

  • Initialised variable to act as the counter – This step occurs only one time but it initialises the main variable that is responsible for how many times the loop will run.
  • A boolean condition involving the initialised variable – This boolean condition will decide when the loop stops running based on an expression involving the earlier initialised variable.
  • An update expression that changes the initialised variable – This part of the FOR loop will update the initialised variable so that it potentially gets closer to meeting the boolean condition.

Here is an example of a FOR loop:

for(int i=1;i<11;i++){
System.out.println("This is loop :"+i);
}

This code shows that the loop is initialised with a variable "i" as 1. It will increment from 1 until the condition is met when it is no longer less than 11, at which point the loop will end.

7.2 WHILE Loops


A WHILE loop is even simpler than a FOR loop. It basically runs as long as a given boolean condition is met.

Here is an example of a WHILE loop:

int counter=1;
while(counter<11){
System.out.println("This is loop: "+counter);
counter++;
}

Here is the result of the code:

Though using a different loop construct the WHILE loop can achieve the same output as the FOR loop. It just has different syntax.

7.3 DO WHILE Loops


The DO WHILE loop is exactly the same as the WHILE loop except for one thing, it evaluates the boolean condition at the end of the block of code rather than at the beginning. This mean the code will always run AT LEAST ONCE before the condition is evaluated.

Here is an example of a DO WHILE loop:

int counter=1;
do{
System.out.println("This is loop: "+counter);
counter++;
}while(counter<11);

The DO WHILE loop also produces the same output but it is highly useful if you KNOW that you want the block of code in the loop to run AT LEAST ONCE.
