Gravity and Velocity

Task 2.1

OK, so now we have a square that can move around taking our input. But we want to do physics simulations, so let’s get started. Let’s create a private float variable called yVelocity in the squareObject. Let us also make one for x. This should hold the velocity of the square in the y direction, and allow us to simulate gravity. Let’s also add another float called gravityStrength to the square that will control how quickly it falls to the ground, and start it at 0.3. 

private float yVelocity=0,xVelocity=0;
private float gravityStrength = 0.05;

Now, let’s add an update method for the square that handles changing its position based on its velocity. Let’s also add a private method to update the velocity based on gravity strength.

public void update()
  {
    updateGravity();
    y+=yVelocity;
   x+=xVelocity;
  }
  private void updateGravity()
  {
    yVelocity+=gravityStrength;
  }

Let’s also call this method within our controller before the square is drawn, so add square.update() just before square.draw().

But we also need to change our code for the asdw keys to change the velocity rather than the position directly, otherwise gravity will take our square downwards forever. So changeY and changeX in the SquareObject class become:

public void changeY(float distance)
  {
    yVelocity+=distance;
  }
public void changeX(float distance)
  {
    xVelocity+=distance;
  }

Be careful when you run it, the changing velocity code from the asdw keys is currently much stronger than gravity!
